# Centro_de_estetica 
Proyecto de la universidad de la materia programacion l, hecho en lenguaje C para la administracion de un centro de estetica 

cliente.h / turno.h: Definición de estructuras y lógica de negocio. 

listaD_Turno.h: Implementación de lista dinámica vinculada (memoria dinámica). 

listaE_Cliente.h: Implementación de lista estática (memoria estatica).
